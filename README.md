Audit (Homepage UX Issues)

The navigation bar feels very plain and doesn’t visually stand out. Since it’s one of the most important elements for guiding users, it should attract attention and make it easy for users to explore different sections. Right now, it blends too much with the rest of the page and doesn’t feel engaging.

The navigation bar is not sticky, which affects the overall user experience. If a user scrolls down and wants to navigate to another section or page, they have to scroll all the way back up. This can feel inconvenient and interrupts the browsing flow.

The overall website feels too white and lacks visual variation. While a clean design is good, too much white space without supporting colors or background elements makes the page feel a bit dull and less engaging. Adding subtle colors or backgrounds could make the experience more visually appealing and encourage users to stay longer.

Design Decisions (Simple Explanation)
Goal
I wanted to design a homepage that is clean, easy to understand, and trustworthy.
The main users are school principals or academic heads, so the design should help them quickly understand what the product does without confusion.

Navigation
I improved the navigation by making it more visible and interactive.
Added boxes around menu items so they stand out
Added hover effect (they slightly grow when hovered)
Added Login and Sign Up buttons separately

This helps users quickly find what they are looking for.

Hero Section
The hero section is the most important part, so I focused on clarity.

Kept the heading simple and readable
Added proper spacing so it doesn’t look crowded
Placed buttons clearly in the center
Added an AI robot image to show what the product is about

This helps users understand the product in a few seconds.

Buttons (Get Started & Demo)
Made buttons easy to see and click
Added hover effect (blue color appears when hovered)
Cursor changes to pointer

This makes the page feel interactive and guides users to take action.

Colors & Design
Used blue as the main color because it feels professional and trustworthy
Added a light background gradient to make the page look modern
Kept everything simple and clean

The goal was not to make it flashy, but clear and professional.

Image Choice
I used an AI robot image because:

It directly represents AI
Makes the page visually interesting
Helps users quickly understand the theme
Product Section

I used simple cards to show products.

This keeps things easy to read and avoids overwhelming the user.

User Thinking
The main user is a busy school principal.
So I made sure:

Information is quick to read
Layout is simple
Important actions are easy to find
Design feels trustworthy
